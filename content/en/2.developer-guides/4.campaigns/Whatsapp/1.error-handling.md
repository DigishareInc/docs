## Common Errors

### ❌ Message Template not found
The sent `message_template_id` does not match any Digishare template (**Text**, **ITM**, or **WABA TM**) configured in your Dashboard.

### ❌ Invalid phone number
The phone number in `wa_id` must be in international format without the `+` sign (e.g., `212600000000`).

### ❌ Media URL not accessible
WhatsApp could not download the media. The URL must be public, not expire quickly, and the file must not exceed size limits (16 MB for video).

### ❌ Missing variable
All variables defined when creating the local template (e.g., `{{1}}`, `{{name}}`) must be provided in the API payload.

::warning
Asynchronous template errors (e.g., rejection by Meta during actual sending) will be notified via your configured Webhooks.
::
