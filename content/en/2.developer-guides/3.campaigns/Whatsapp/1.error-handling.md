## 🚀 Quick Reference (Common Errors)

| Code | Label | Meaning | Action |
| :--- | :--- | :--- | :--- |
| **131049** | `ecosystem_limit` | Per-user marketing frequency limit hit. | Wait 24h. |
| **131047** | `chat_expired` | Window > 24h since last user reply. | Send a **Template**. |
| **131026** | `undeliverable` | Blocked, Not on WA, or restricted. | Verify number. |
| **2007** | `policy_violated` | Content violates Meta's formatting policies. | Review [Meta Policies](https://business.whatsapp.com/policy). |
| **429** | `rate_limited` | API throughput has been reached. | Implement **Backoff**. |

## 🌩️ Detailed Error Codes

### 1. Integrity & Compliance

| Code | Description | Potential Solution |
| :--- | :--- | :--- |
| **368** | Temporary Block | WABA restricted for policy violations. Check Business Support. |
| **130497** | Country Restricted | Messaging a restricted/sanctioned region. |
| **131031** | Account Locked | Resolve by checking Health Status or 2FA. |

### 2. Messaging & Delivery

| Code | Description | Insight |
| :--- | :--- | :--- |
| **1001** | Message Too Long | Body exceeds character limit (usually 4096 chars). |
| **1006** | Resource Not Found | Recipient or media object not found on WA servers. |
| **131009** | Invalid Parameter | One or more payload values are malformed. |
| **131021** | Self-Messaging | Sender and recipient are identical. |
| **131052** | Media Download Failed | Meta cannot retrieve your media URL. |
| **131053** | Media Upload Failed | Unsupported media type or server failure. |

### 3. Template Management

| Code | Description | Key Insight |
| :--- | :--- | :--- |
| **2005** | Translated Text Too Long | Variables + Template text exceeds UI limits. |
| **132000** | Param Count Mismatch | Variables provided != Variables in template. |
| **132001** | Template Not Found | ID/Name not found or not approved. |
| **132016** | Template Paused | High report rate; template temporarily disabled. |

## 🏗️ Gateway & Infrastructure

| Code | Label | Potential Solution |
| :--- | :--- | :--- |
| **400** | Payment Issue | Overdue invoices or currency/timezone not set. |
| **401** | Invalid API Key | Ensure `WA-API-KEY` (Meta Token) is correct. |
| **502** | Gateway Unavailable | API gateway is down or in maintenance. |
| **131057** | Meta Maintenance | WABA is undergoing a throughput upgrade. |
