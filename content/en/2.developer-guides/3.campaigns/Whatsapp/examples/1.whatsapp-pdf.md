This guide explains how to send a WhatsApp campaign including a **PDF document**.

::note

**Prerequisites**:
- Valid **Bearer Token** in Authorization header.
- **Digishare Message Template** (ITM or **WABA TM**) configured to accept a header with PDF link.
- Publicly accessible PDF URL.
::

## 1. Payload Requirements

To send a personalized PDF, you must provide the header link and recipient information in the `global_data` and `recipients` blocks.

### Required Fields
- `wa_id`: Recipient's phone number (in `global_data` as template variable).
- `header.link`: Publicly accessible URL of the PDF.
- `tags`: Optional array for categorizing campaigns.
- `conversation_id`: Optional identifier for conversation tracking.

## 2. API Implementation (Personalized PDF)

To send a unique, personalized PDF to each recipient (e.g., individual customer invoices), map the header link dynamically in `global_data` and specify the specific file URL for each recipient inside the `recipients` array:

::api-playground
---
method: POST
url: https://api.digishare.ma/v1/campaigns
description: Send unique, personalized PDF documents to each recipient.
variables:
  token: YOUR_TOKEN
  senderId: YOUR_SENDER_ID
  templateId: YOUR_TEMPLATE_ID
headers:
  Authorization: Bearer {token}
  Content-Type: application/json
body:
  title: Personalized Invoice Distribution
  channel: whatsapp
  sender_label_id: "{senderId}"
  message_template_id: "{templateId}"
  global_data:
    wa_id: "%recipient.wa_id%"
    header:
      link: "%recipient.pdf_url%"
  recipients:
    - wa_id: "212600000000"
      pdf_url: "https://your-site.com/invoices/inv_john.pdf"
    - wa_id: "212600000001"
      pdf_url: "https://your-site.com/invoices/inv_jane.pdf"
  tags:
    - "campaign"
  conversation_id: "message-out"
---
::

## 3. Alternative: Handling Static PDFs

If your campaign uses the **same** PDF document for all recipients (e.g., a shared product catalog or generic flyer), you don't need to define the URL per recipient. Simply define the static link once directly in `global_data`:

```json
{
  "title": "Monthly Catalog",
  "channel": "whatsapp",
  "sender_label_id": "your-sender-label-id",
  "message_template_id": "your-message-template-id",
  "global_data": { 
    "wa_id": "%recipient.wa_id%",
    "header": {
      "link": "https://your-site.com/catalog.pdf"
    }
  },
  "recipients": [
    { "wa_id": "212600000000" },
    { "wa_id": "212600000001" }
  ],
  "tags": ["campaign"]
}
```
