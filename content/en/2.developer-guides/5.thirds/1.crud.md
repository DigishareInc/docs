## List Thirds

Retrieve all contacts with pagination and filtering.

::api-playground{method="GET" url="https://api.digishare.ma/v1/thirds" description="Get a paginated list of contacts." :variables='{"token": "YOUR_ACCESS_TOKEN"}' :headers='{"Authorization": "Bearer {token}"}'}
::

### Parameters

| Parameter  | Description                                |
| ---------- | ------------------------------------------ |
| `page`     | Page number (default: 1)                   |
| `limit`    | Items per page (default: 15, max: 100)     |
| `search`   | Search by name, email, or phone            |
| `orderBy`  | Sort by field (e.g., `created_at`)         |
| `sortedBy` | Sort direction: `asc` or `desc`            |
| `include`  | Load relations: `tags`, `groups`, `source` |

### Example Response

```json
{
  "data": [
    {
      "id": "xYz123AbC",
      "first_name": "Sarah",
      "last_name": "Connor",
      "email": "sarah@example.com",
      "mobile": "+212612345678"
    }
  ],
  "meta": {
    "pagination": {
      "total": 150,
      "per_page": 15,
      "current_page": 1,
      "total_pages": 10
    }
  }
}
```

## Get a Third

Retrieve a single contact by ID.

::api-playground{method="GET" url="https://api.digishare.ma/v1/thirds/{id}" description="Get a specific contact." :variables='{"id": "THIRD_ID", "token": "YOUR_ACCESS_TOKEN"}' :headers='{"Authorization": "Bearer {token}"}'}
::

### Parameters

| Parameter | Description                                                            |
| --------- | ---------------------------------------------------------------------- |
| `include` | Load relations: `tags`, `groups`, `source`, `tickets`, `conversations` |

```json
      "company_name": "Cyberdyne",
      "title": "Director"
    },
    "created_at": "2026-01-15T10:30:00Z"
  }
}
```

## Create a Third

Add a new contact to your database.

::api-playground{method="POST" url="https://api.digishare.ma/v1/thirds" description="Create a new contact." :variables='{"token": "YOUR_ACCESS_TOKEN"}' :headers='{"Authorization": "Bearer {token}", "Content-Type": "application/json"}' :body='{"first_name": "John", "last_name": "Doe", "email": "john@example.com", "mobile": "+212612345678"}'}
::

### Request Body

```json
{
  "first_name": "John",
  "last_name": "Doe",
  "email": "john@example.com",
  "mobile": "+212612345678",
  "wa_id": "212612345678",
  "lang": "en",
  "tags": ["lead", "webinar"],
  "data": {
    "company_name": "Acme Inc",
    "title": "Manager"
  }
}
```

### Available Fields

**Basic Info:** `first_name`, `last_name`, `name`, `email`, `mobile`, `phone`, `wa_id`

**Profile:** `address`, `address_2`, `lang`, `gender`, `birthday`

**Flags:** `is_company`, `premium`, `verified`, `status`

**Relations:** `source_id`, `country_id`, `state_id`, `city_id`, `parent_id`

**Arrays:** `tags`, `group_type_id`

**Extended:** `data` object (see [Overview](/developer-guides/thirds/overview#extended-data))

::note
**Automatic company assignment.** The `company_id` is automatically set from your authentication token. You don't need to pass it.
::

::tip
**Phone numbers are auto-formatted.** We standardize `mobile`, `phone`, and `wa_id` to international E.164 format.
::

### Response

Returns the created contact with `201 Created`.

## Update a Third

Modify an existing contact. Only include fields you want to change.

::api-playground{method="PATCH" url="https://api.digishare.ma/v1/thirds/{id}" description="Update a contact." :variables='{"id": "THIRD_ID", "token": "YOUR_ACCESS_TOKEN"}' :headers='{"Authorization": "Bearer {token}", "Content-Type": "application/json"}' :body='{"premium": true, "verified": true}'}
::

### Request Body

```json
{
  "premium": true,
  "verified": true,
  "data": {
    "title": "Senior Director"
  }
}
```

::note
**Partial updates.** Fields not included in the request remain unchanged.
::

::warning
**Group replacement.** The `group_type_id` array replaces all existing groups. Include current groups to keep them.
::

### Response

Returns the updated contact with `200 OK`.

---

## Delete a Third

Remove a contact from your database.

::api-playground{method="DELETE" url="https://api.digishare.ma/v1/thirds/{id}" description="Delete a contact." :variables='{"id": "THIRD_ID", "token": "YOUR_ACCESS_TOKEN"}' :headers='{"Authorization": "Bearer {token}"}'}
::

::warning
**Soft delete.** The contact is marked as deleted but can be restored by an admin.
::

### Response

Returns `204 No Content` on success.

## Error Handling

### Common Errors

| Status | Meaning                  |
| ------ | ------------------------ |
| `401`  | Invalid or missing token |
| `403`  | Insufficient permissions |
| `404`  | Contact not found        |
| `422`  | Validation failed        |

### Validation Error Example

```json
{
  "message": "The given data was invalid.",
  "errors": {
    "wa_id": ["This WhatsApp ID is already in use."],
    "ext_ref": ["The external reference must be unique."]
  }
}
```
