Custom Fields allow you to extend your Third entities with dynamic, flexible metadata. Use them to store statistics, preferences, or any custom data on your contacts.

---

## 1. Define Custom Fields

Before storing data, optionally define the field schema to ensure consistent data types.

### Create a Field Definition

::api-playground{method="POST" url="https://api.digishare.ma/v1/custom-field-definitions" description="Create a new custom field definition." :headers='{"Authorization": "Bearer {token}", "Content-Type": "application/json"}' :body='{"key": "balance", "label": "Account Balance", "type": "float"}'}
::

### Field Definition Parameters

| Parameter | Type   | Required | Description                                             |
| --------- | ------ | -------- | ------------------------------------------------------- |
| `key`     | string | Yes      | Unique field identifier                                 |
| `label`   | string | Yes      | Human-readable display name                             |
| `type`    | string | Yes      | Data type: `string`, `int`, `float`, `bool`, `date`     |
| `options` | object | No       | Additional configuration (dropdown options, validation) |

### Supported Data Types

| Type     | Description     | Example Value        |
| -------- | --------------- | -------------------- |
| `string` | Text values     | `"Premium Customer"` |
| `int`    | Whole numbers   | `42`                 |
| `float`  | Decimal numbers | `150.75`             |
| `bool`   | True/False      | `true`               |
| `date`   | ISO date string | `"2026-01-15"`       |

## 2. List Field Definitions

Retrieve all defined custom fields:

::api-playground{method="GET" url="https://api.digishare.ma/v1/custom-field-definitions" description="List all custom field definitions." :headers='{"Authorization": "Bearer {token}", "Content-Type": "application/json"}'}
::

### Response Example

```json
{
  "data": [
    {
      "key": "balance",
      "label": "Account Balance",
      "type": "float"
    },
    {
      "key": "order_count",
      "label": "Total Orders",
      "type": "int"
    }
  ]
}
```

## 3. Set Custom Field Values

Store custom data on a Third entity:

::api-playground{method="POST" url="https://api.digishare.ma/v1/thirds/{hashed_id}/custom-fields" description="Update custom fields for a Third." :variables='{"hashed_id": "YOUR_THIRD_ID", "token": "YOUR_ACCESS_TOKEN"}' :headers='{"Authorization": "Bearer {token}", "Content-Type": "application/json"}' :body='{"fields": {"balance": 150.75, "order_count": 12, "is_vip": true}}'}
::

::note
**Merge behavior:** This endpoint **merges** the provided fields with existing data. Fields not included in the request remain unchanged. To remove a field, use the [Delete endpoint](#5-delete-custom-fields).
::

### Request Body

| Parameter | Type   | Required | Description                   |
| --------- | ------ | -------- | ----------------------------- |
| `fields`  | object | Yes      | Key-value pairs of field data |

### Example

```bash
POST /v1/thirds/{hashed_id}/custom-fields
Content-Type: application/json

{
  "fields": {
    "balance": 150.75,
    "order_count": 12,
    "is_vip": true,
    "loyalty_tier": "gold"
  }
}
```

## 4. Get Custom Field Values

Retrieve custom fields for a Third:

::api-playground{method="GET" url="https://api.digishare.ma/v1/thirds/{hashed_id}/custom-fields" description="Get all custom fields for a Third." :variables='{"hashed_id": "YOUR_THIRD_ID", "token": "YOUR_ACCESS_TOKEN"}' :headers='{"Authorization": "Bearer {token}", "Content-Type": "application/json"}'}
::

### Response Example

```json
{
  "data": {
    "balance": 150.75,
    "order_count": 12,
    "is_vip": true,
    "loyalty_tier": "gold"
  }
}
```

---

## 5. Delete Custom Fields

Remove custom fields from a Third:

::api-playground{method="DELETE" url="https://api.digishare.ma/v1/thirds/{hashed_id}/custom-fields" description="Delete custom fields from a Third." :variables='{"hashed_id": "YOUR_THIRD_ID", "token": "YOUR_ACCESS_TOKEN"}' :headers='{"Authorization": "Bearer {token}", "Content-Type": "application/json"}' :body='{"keys": ["balance", "order_count"]}'}
::

### Delete Specific Fields

```bash
DELETE /v1/thirds/{hashed_id}/custom-fields
Content-Type: application/json

{
  "keys": ["balance", "order_count"]
}
```
