Ce guide montre comment envoyer une campagne WhatsApp incluant une **image**.

::note
**Prérequis** :
- **Jeton d'accès (Bearer Token)** valide.
- **Modèle de Message Digishare** (type ITM ou **WABA TM**) configuré pour accepter une image.
::

## 1. Préparation du Payload

Pour envoyer une image, vous devez fournir l'URL publique de l'image (JPG ou PNG) dans le champ `header_url`.

::api-playground{method="POST" url="https://api.digishare.ma/v1/campaigns" description="Envoyer une image personnalisée." :variables='{"token": "VOTRE_TOKEN", "senderId": "VOTRE_ID_EXPEDITEUR", "templateId": "VOTRE_ID_MODELE"}' :headers='{"Authorization": "Bearer {token}", "Content-Type": "application/json"}' :body='{"title": "Promo Produit", "channel": "whatsapp", "sender_label_id": "{senderId}", "message_template_id": "{templateId}", "recipients": [{"wa_id": "212600000000", "header_url": "https://votre-site.ma/produit-123.jpg"}] }'}
::

## 2. Utilisation de `global_data` pour l'image

Si l'image est identique pour tout le monde, utilisez `global_data` pour simplifier votre payload.

```json
{
  "title": "Annonce Soldes",
  "global_data": { 
    "header_url": "https://site.ma/soldes_ete.png" 
  },
  "recipients": [
    { "wa_id": "212600000000" },
    { "wa_id": "212600000001" }
  ]
}
```
