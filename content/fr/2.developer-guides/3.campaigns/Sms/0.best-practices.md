## 1. Limites de Caractères et Encodage SMS

La compréhension de l'encodage SMS est essentielle pour maîtriser les coûts de campagne et éviter le découpage inattendu des messages.

### Encodage GSM 7 bits (Texte Standard)

Les caractères latins de base, les chiffres et la ponctuation courante utilisent l'encodage **GSM 7 bits** :

- **SMS Simple** : Jusqu'à **160 caractères**.
- **SMS Multi-parties** : Les messages dépassant 160 caractères sont découpés en segments de **153 caractères** chacun (7 caractères réservés aux en-têtes de concaténation).

### Encodage Unicode / UCS-2 (Caractères Spéciaux & Emojis)

Si votre message contient des accents (`é`, `à`), des alphabets non latins (Arabe `مرحبا`, Cyrillique, etc.) ou des emojis (`🚀`) :

- **SMS Simple** : Jusqu'à **70 caractères**.
- **SMS Multi-parties** : Les messages dépassant 70 caractères sont découpés en segments de **67 caractères** chacun.

::tip
**Conseil d'optimisation** : Évitez les emojis ou les caractères non-GSM dans les campagnes marketing à grand volume pour maintenir la limite par segment à 160 caractères au lieu de 70.
::

## 2. Formatage des Numéros de Téléphone (Norme E.164)

Formatez toujours les numéros de téléphone des destinataires selon la **norme E.164** sans espaces, tirets ni zéros initiaux :

- **Correct** : `"212671791198"` ou `"+212671791198"`
- **Incorrect** : `"0671791198"`, `"212 671 791 198"`, `"+212 (0)671791198"`

```json
{
  "recipients": [
    { "mobile": "212671791198" },
    { "mobile": "212661234567" }
  ]
}
```

::note
L'API Digishare normalise automatiquement les numéros nationaux valides, mais la transmission de chaînes au format E.164 garantit un traitement plus rapide et évite les échecs d'acheminement.
::

## 3. Envoi par Lots à Haut Volume

Pour un débit API et une efficacité backend optimaux :

- Regroupez les destinataires par lots allant jusqu'à **1 000 éléments par requête API** dans le tableau `recipients`.
- Évitez les requêtes API individuelles répétées pour chaque destinataire lors de l'envoi de campagnes de diffusion.

---

## 4. Conformité et Désabonnement

- **Nom d'Expéditeur (Sender Label)** : Assurez-vous que votre `sender_label_id` est enregistré et approuvé dans votre région.
- **Instructions de Désabonnement** : Pour les messages promotionnels, incluez les mentions légales de désabonnement requises par la réglementation locale (ex. *"STOP au 36000"*).
- **Plages Horaires** : Respectez la réglementation locale concernant les heures d'interdiction des SMS marketing (ex. éviter les envois tardifs la nuit).
